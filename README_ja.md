# Learning English 🇬🇧
![Learning English Banner](jjj.png)
良質な英語学習リソースを厳選したコレクションで、リスニング、スピーキング、リーディング、ライティングなどのコアスキルの向上に焦点を当てています。文法、語彙、メディアリソースを含み、英語学習をより効果的にサポートします。
対象:

- すべてのレベルの英語学習者
- 体系的に練習を求める独習者
- 教材を探している教育者
- 仕事、学習、旅行のため英語力向上が必要なプロフェッショナル
- プロジェクトは継続的に更新され、適切な良いリソースがあれば随時追加されます！

### ポッドキャストリソース 🎙️
